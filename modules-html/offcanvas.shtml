Off canvas menu

Beskrivelse og afhængigheder

Modulet er sat op med aria attributer og sætter tabindex="-1" på menupunkterne når menuen er lukket og tabindex="0" når den er åben. Det sørger for at man ikke tab'er ind i menuen når den er lukket.

Når man tab'er igennem menupunkterne, vil .offcanvas-toggler få focus efter sidste menupunkt og man starter derefter forfra. Det sørger for at man ikke tab'er ud af menuen når den er åben.

Modulet laver selv et overlay, som lukker menuen når man klikker på det.

Når menuen er åben, sættes der en klasse på body for at give den overflow: hidden. Dette er for at man ikke kan scrolle siden.

Javascript initialization eksempel:

$(".offcanvas-menu").each(function () { var offcanvas = new dis.OffCanvasMenu({ container: this }); });

Basic html

<div class="offcanvas-menu"> <button class="offcanvas-toggler" title="navigation menu" aria-label="navigation menu">Menu</button> <ul> <li> <a href="#1">Menupunkt nummer 1</a> </li> <li> <a href="#2">Menupunkt nummer 2</a> </li> </ul> </div>

Filer der skal bruges

dis.module.offcanvas.js, dis.module.offcanvas.scss, dis.module.offcanvas.shtml
