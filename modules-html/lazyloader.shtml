lazyLoader

Beskrivelse og afhængigheder

Dette modul er lavet som et alternativ til imageHandler'en, hvor billeder først hentes idet de er synlige i viewporten.
initialiseres ved at kalde setupLazyImages()
Eks.:

setupLazyImages(".lazy-img", 200); // @param1: Selector - class på lazy-images // @param2: BufferMargin - Antal pixels før og efter viewporten billederne skal loade. // (Gør det muligt at preloade billederne lige inden brugeren ser dem)

Filer der skal bruges

js/utils/lazyloader.js
js/utils/classList.js
scss/Modules/lazyLoader.scss (optionel)
lazyLoader HTML:

<div class="lazy-img" data-src-xs="http://placehold.it/480x270/ffaa00/ffffff/?text=Lazy480x270" data-src-sm="http://placehold.it/768x432/ffaa00/ffffff/?text=Lazy768x432" data-src-md="http://placehold.it/1280x720/ffaa00/ffffff/?text=Lazy1280x720" data-alt="Alt text"> <noscript> <img src="http://placehold.it/480x270/ffaa00/ffffff/?text=NoScriptLazyImage" alt="Alt text"> </noscript> </div>
Eller som baggrundsbillede
<div class="lazy-img lazy-bg" data-src-xs="http://placehold.it/480x270/ff3300/ffffff/?text=Lazy480x270" data-src-sm="http://placehold.it/768x432/ff3300/ffffff/?text=Lazy768x432" data-src-md="http://placehold.it/1280x720/ff3300/ffffff/?text=Lazy1280x720"> </div>
