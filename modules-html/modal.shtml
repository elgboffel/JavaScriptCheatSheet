Modals

Description and dependencies

Class-based modal with unlimited instantiation ability and flexible setup. Main class ( modal) has a lot of options and methods, while two helper functions ( alert() and confirm, so you may want to alias them upon import) gives you shortcuts for displaying confirms and alerts without a lot of setup.

Files you need:

  • modal.js
  • modal.scss (contains basic stuff - for theming, consider using the modifierClass option in the modal options)
